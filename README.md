<h1> Challenge del amigo secreto </h1>

<img src="https://img.shields.io/badge/STATUS-FINALIZADO-green" />

- Descripción del proyecto: El usuario debe añadir los nombres de los amigos que desee, y al realizar el sorteo se mostrará de manera aleatoria uno de ellos como su amigo secreto.

- Tecnología utilizada: JavaScript.

- Autor: Gerardo Salfate.
