## 📌 Descripción del proyecto

Esta aplicación es una solución interactiva desarrollada para automatizar el sorteo del "Amigo Secreto", elaborada durante el programa **Oracle Next Education (ONE)** de **Oracle + Alura Latam**. El sistema permite a los usuarios ingresar nombres de participantes de forma dinámica, gestionando una lista visual y realizando un sorteo aleatorio con validaciones integradas para asegurar una experiencia de usuario fluida.

## 🚀 Conceptos de programación aplicados

El proyecto pone en práctica pilares fundamentales del desarrollo web y la lógica de programación:

* **Gestión dinámica de datos:** Captura y almacenamiento de nombres mediante el uso de arreglos (arrays).
* **Validación de entradas:** Control de errores para evitar campos vacíos y garantizar la integridad de la lista.
* **Algoritmos de aleatoriedad:** Implementación de lógica matemática para seleccionar un ganador de forma imparcial.
* **Sincronización con el DOM:** Renderizado automático de la lista de amigos y el resultado del sorteo en la interfaz HTML.
